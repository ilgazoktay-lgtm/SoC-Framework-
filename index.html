VDT is computed as Recovered Unit Value / eDiM (no normalization).
Final Score is computed as Base Model Result × VDT × AMG × HCI × VVI.
CSI = eDiM × Cost-per-second / Recovered Unit Value.
Margin Buffer Ratio = (Recovered Unit Value − Cost-per-second×eDiM) / Recovered Unit Value.
